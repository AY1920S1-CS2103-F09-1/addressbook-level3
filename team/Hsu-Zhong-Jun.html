PROJECT: Scheduler

Overview

Scheduler is a desktop application which is used to automate the process of scheduling interviews. Its main feature is matching interviewees with interview time slots to generate an interview timetable. The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 10 kLoC.
